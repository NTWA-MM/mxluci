# mxluci

A Kotlin-based project for managing and orchestrating workflows.

## Getting Started

### Prerequisites
- Kotlin 1.8+
- Java 11+
- Gradle 7.0+

### Build
```bash
./gradlew build
```

### Run Tests
```bash
./gradlew test
```

## Project Structure
- `src/main/kotlin/` - Source code
- `src/test/kotlin/` - Test code
